## 1. 基本操作

### 進入和退出

| 操作       | 按鍵          | 說明           |
| ---------- | ------------- | -------------- |
| 退出       | `:q!`         | 放棄更改並退出 |
| 保存並退出 | `:wq` 或 `ZZ` | 保存更改並退出 |
| 另存為     | `:w filename` | 保存為新文件   |
| 強制退出   | `:q` + Enter  | 無更改時退出   |

### 在 PTY MCP 中的退出技巧

如果在 Vim 中卡住,按 `Esc` 確保進入 Normal 模式,然後:

```
:wq        # 保存退出
:q!        # 不保存強制退出
```

## 2. 模式切換

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   ┌──────────┐         ┌──────────┐         ┌──────────┐   │
│   │  Normal  │ ──i──▶  │  Insert  │ ──Esc──▶│  Normal  │   │
│   │   Mode   │◀──a──   │   Mode   │◀──a──   │   Mode   │   │
│   └──────────┘         └──────────┘         └──────────┘   │
│                                                             │
│   ┌──────────┐         ┌──────────┐                        │
│   │ Command  │ ──:──▶  │  Ex      │                        │
│   │   Line   │◀─Enter─ │  Mode    │                        │
│   └──────────┘         └──────────┘                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| 按鍵  | 功能                       |
| ----- | -------------------------- |
| `i`   | 在當前字符前插入           |
| `a`   | 在當前字符後插入           |
| `o`   | 在下方新建行並進入插入模式 |
| `O`   | 在上方新建行並進入插入模式 |
| `I`   | 在行首插入                 |
| `A`   | 在行尾插入                 |
| `Esc` | 返回 Normal 模式           |
| `:`   | 進入命令行模式             |

## 5. 搜索和替換

### 搜索

| 按鍵       | 功能               |
| ---------- | ------------------ |
| `/pattern` | 向下搜索           |
| `?pattern` | 向上搜索           |
| `n`        | 下一個匹配         |
| `N`        | 上一个匹配         |
| `*`        | 搜索當前單詞(向下) |

### 替換

| 命令             | 功能                 |
| ---------------- | -------------------- |
| `:s/old/new`     | 替換當前行第一個匹配 |
| `:s/old/new/g`   | 替換當前行所有匹配   |
| `:%s/old/new/g`  | 替換文件中所有匹配   |
| `:%s/old/new/gc` | 替換並確認每個匹配   |

### 正則表達式

| 模式    | 說明       |
| ------- | ---------- |
| `.`     | 任意單字符 |
| `*`     | 零或多個   |
| `\+`    | 一或多個   |
| `\?`    | 零或一     |
| `^`     | 行首       |
| `$`     | 行尾       |
| `[abc]` | 字符集     |

## 8. 常用配置

### .vimrc 基礎配置

```vim
" 顯示行號
set number

" 顯示相對行號
set relativenumber

" 開啟語法高亮
syntax on

" 設置縮進
set tabstop=4
set shiftwidth=4
set softtabstop=4
set expandtab

" 開啟智能縮進
set smartindent

" 顯示匹配的括號
set showmatch

" 開啟搜索高亮
set hlsearch

" 搜索時忽略大小寫
set ignorecase
set smartcase

" 開啟自動縮進
set autoindent

" 顯示當前光標位置
set cursorline

" 設置字符編碼
set encoding=utf-8

" 開啟鼠標支持(如果終端支持)
set mouse=a

" 顯示底部命令
set showcmd

" 開啟文件類型檢測
filetype plugin indent on
```

## 9. 常見問題

### Q1: 在 PTY MCP 中如何使用方向鍵?

在 Normal 模式下,可以使用:

- `h`, `j`, `k`, `l` 代替方向鍵
- 或者使用 `\x1b[A` 等轉義序列

### Q2: 如何處理特殊字符?

使用 `write_bytes` 工具:

```python
# 上箭頭: \x1b[A
# 下箭頭: \x1b[B
# 右箭頭: \x1b[C
# 左箭頭: \x1b[D
```

### Q3: 如何復制文本?

1. 在 Visual 模式 (`v`) 選擇文本
2. 按 `y` 複製
3. 按 `p` 粘貼

### Q4: 如何處理大文件?

使用以下技巧:

- `gg` 快速跳到開頭
- `G` 快速跳到結尾
- `Ctrl+d` / `Ctrl+u` 快速滾動
- `/pattern` 搜索定位

### Q5: 如何保存和退出?

確保在 Normal 模式 (按 `Esc`):

- `:w` - 保存
- `:q` - 退出
- `:wq` 或 `ZZ` - 保存並退出
- `:q!` - 不保存強制退出

### Q6: Vim 無響應怎麼辦?

1. 嘗試按 `Esc` 幾次
2. 檢查是否在 Insert 模式
3. 使用 `:q!` 強制退出

## 實用技巧

### 1. 快速註釋多行

1. `Ctrl+v` 進入可視塊模式
2. 選擇行
3. `I` 進入插入模式
4. 輸入註釋字符 (如 `//` 或 `#`)
5. `Esc` 完成

### 2. 批量替換

```vim
:%s/old/new/gc    " 逐個確認替換
:%s/old/new/g     " 全部替換
:5,10s/old/new/g  " 替換第5-10行
```

### 3. 快速移動到行尾

在 Normal 模式下:

- `$` - 移動到行尾
- `A` - 移動到行尾並進入插入模式

### 4. 重複上次操作

- `.` - 重複上次編輯操作
- `n.` - 執行 n 次上次編輯

### 5. 快速退出插入模式

- `jj` (如果映射了)
- `Escape`
- `Ctrl+c` (不觸發 Abbrev)

## PTY MCP 專用技巧

### 使用流程

1. **進入 Vim**: `vim filename` 或 `vim`
2. **編輯文件**: 按 `i` 進入插入模式
3. **保存**: 按 `Esc` 回到 Normal 模式,輸入 `:w`
4. **退出**: `:q` 或 `:wq`

### 截圖確認狀態

在操作之間使用 `screenshot` 工具確認:

- 是否在正確模式
- 是否進入插入模式
- 是否保存成功

### 處理長命令

長命令可以拆分成多次輸入:

```python
write(text="set number", with_enter=True)
write(text="syntax on", with_enter=True)
```
