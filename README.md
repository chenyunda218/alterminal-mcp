# Alterminal

一個基於 MCP (Model Context Protocol) 的 PTY 終端服務器，為 AI Agent 提供交互式終端能力。

## 概述

Alterpty 是一個 MCP Server，它啟動一個 bash 偽終端 (PTY)，並通過 MCP 協議暴露一系列工具，讓 AI 能夠：

- 讀取終端輸出
- 向終端發送命令
- 獲取終端狀態

## 功能特色

- 🚀 **即時截圖**：獲取當前終端狀態的文本快照
- 📝 **命令輸入**：支持文本和原始字節輸入
- 🎯 **精確定位**：查詢和控制光標位置
- 📐 **尺寸查詢**：獲取終端大小（行/列）
- 🔄 **VT100 解析**：完整的 VT100 終端轉義序列解析

## 工具列表

| 工具名稱      | 描述                       | 參數                     |
| ------------- | -------------------------- | ------------------------ |
| `screenshot`  | 獲取當前終端狀態的文本截圖 | 無                       |
| `get_size`    | 獲取終端尺寸               | 無                       |
| `write`       | 向 PTY 寫入文本            | `text` (string, 必填)    |
| `write_bytes` | 向 PTY 寫入字節序列        | `bytes` (number[], 必填) |
| `get_cursor`  | 獲取當前光標位置           | 無                       |

## 快速開始

### 安裝

```bash
git clone https://github.com/alterminal/alterpty.git
cd alterpty
go build -o alterpty .
```

### 運行

```bash
./alterpty
```

服務器會通過 STDIN/STDOUT 與 MCP 客戶端通信。
